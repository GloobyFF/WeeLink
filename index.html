Your gaming PC, placed inside visionOS.

WeeLink joins a Windows launcher, NVIDIA CloudXR 6, and an Apple Vision Pro client into one local-network streaming workflow.

Vision Pro first

Native foveated streaming path

Built around Apple's FoveatedStreaming framework on visionOS 26.4+, with CloudXR handling the low-latency PC encode pipeline.

Game library

Steam, Epic, Xbox, OpenXR VR, UEVR

Auto-discover games, mark OpenXR VR and UEVR titles, add custom executables, create collections, and launch directly from the headset.

Desktop mode

High-quality Windows capture

Stream any monitor with WGC capture, sRGB-correct downsampling, sharpness controls, FPS status, automatic aspect-ratio handling, curved screen mode, and passthrough display mode.

Stereo media

SBS and top-bottom 3D

Watch 3D Blu-ray rips, YouTube 3D, 3DS emulators, and SuperDepth3D output on a fixed virtual theater screen.

Multi-monitor

Bring your whole desk into space

Mirror multiple displays as independent panels with size, position, tilt, yaw, layout presets, and Windows layout replication.

Control API

Remote launch and live session control

HTTPS and WebSocket APIs let the Vision Pro client browse games, switch monitors, toggle 3D, trigger hotkeys, and track status.

System requirements

WeeLink is built for a strong local network and an NVIDIA RTX PC running modern Windows and OpenXR content.

PC GPUNVIDIA RTX series GPU with CloudXR 6 encoder support.
PC OSWindows 10 22H2 or later. Windows 11 recommended.
HeadsetApple Vision Pro on the same subnet as the Windows host.
visionOSvisionOS 26.4 or later for Apple's FoveatedStreaming path.
NetworkEthernet for PC plus Wi-Fi 6 or Wi-Fi 6E for the headset.

Support

Need help with WeeLink setup, pairing, purchases, or streaming issues? Contact support and include enough detail for us to reproduce the problem.

Email Support GitHub Issues Privacy Policy

What to include

  • Your Apple Vision Pro model and visionOS version.
  • Your Windows version, NVIDIA GPU model, and WeeLink version.
  • A short description of the issue, screenshots, and any error messages.
  • For purchase or membership questions, include the order date and the Apple ID email domain only, not your full password or payment details.

We usually respond within 1-3 business days.
