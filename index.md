# WeeLink

Windows launcher for native PCVR streaming to Apple Vision Pro.

## Download

The current test build is available as a draft GitHub Release.

> This is an early test build. Do not publish it publicly until NVIDIA CloudXR
> redistribution rights are confirmed.

## Requirements

- Windows 11
- NVIDIA GPU
- SteamVR / OpenXR-compatible PCVR content
- Apple Vision Pro client on the same local network

## Installer

Current installer file:

`VlinkSetup-0.5.0.exe`

The installer filename will be updated in a future release. The product name is
now WeeLink.
